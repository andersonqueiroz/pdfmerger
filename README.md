# PDF Merger

Script criado para agrupar arquivos PDF em um único documento. A pesquisa é feita recursivamente a partir do diretório raiz do executável.

## Exemplos de uso

Para mesclar os arquivos no diretório do script:

    python pdfmerger

Para mesclar os arquivos a partir de um diretório específico:

    python pdfmerger /caminho/do/diretorio
